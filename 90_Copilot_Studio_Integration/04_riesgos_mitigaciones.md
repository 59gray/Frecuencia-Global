# Riesgos y Mitigaciones

| Riesgo                                         | Mitigación Propuesta                         |
|------------------------------------------------|----------------------------------------------|
| Incompatibilidad de scripts/automatizaciones   | Rediseño en Power Automate o Power Platform  |
| Pérdida de contexto en prompts/documentación   | Validación manual y pruebas de usuario       |
| Falta de soporte para flujos n8n               | Migración manual a Power Automate            |
| Activos visuales no migrables                  | Mantener en repositorio externo              |
| Dependencias externas no soportadas            | Documentar y buscar alternativas             |
| Campos vacíos en Copilot Studio                | Preparar contenido listo para pegar           |
