# Frecuencia Global

**Análisis internacional con pulso electrónico.**

Marca multiplataforma que traduce geopolítica, relaciones internacionales y cultura global a formatos accesibles con identidad visual inspirada en música electrónica.

---

## Estructura del repositorio

```text
system/                          -> Sistema operativo de agentes
|-- SISTEMA_MAESTRO.md           -> Documento central (empieza aqui)
|-- agents/                      -> Agentes core, soporte y especialidad
|-- playbooks/                   -> Procedimientos operativos
|-- templates/                   -> Plantillas base + paquete formal de video
|-- rules/                       -> Reglas del sistema
|-- workflows/                   -> Mapa de flujos de produccion
`-- roadmap/                     -> Plan de desarrollo

01_Estrategia/                   -> Documentos estrategicos fundacionales
02_Brand_System/                 -> Kit de marca operativo
03_Editorial/                    -> Guias editoriales por pilar
04_Produccion/                   -> Flujos de produccion y ejemplos reales
05_Monetizacion/                 -> Estrategia de monetizacion
06_Assets/                       -> Assets consolidados (base, v1-v6, podcast, host)
07_Operaciones/                  -> Guias operativas y reportes por plataforma
08_n8n/                          -> Workflows de automatizacion (n8n cloud)
website/                         -> Sitio Astro (Vercel)
static/                          -> Tipografias locales
scripts/                         -> Automatizaciones sociales y utilidades
```

## Para empezar

1. Lee [`system/SISTEMA_MAESTRO.md`](system/SISTEMA_MAESTRO.md) para la visión completa del sistema.
2. Consulta los agentes en [`system/agents/`](system/agents/) para ver responsabilidades y handoffs.
3. Sigue los playbooks en [`system/playbooks/`](system/playbooks/) para ejecutar el flujo correcto.
4. Usa las plantillas en [`system/templates/`](system/templates/) como formato estándar de trabajo.
5. Respeta las reglas en [`system/rules/`](system/rules/) antes de producir o exportar cualquier pieza.
6. Para video, parte por [`04_Produccion/FG_Flujo_Produccion_Video.md`](04_Produccion/FG_Flujo_Produccion_Video.md) y el [`system/agents/AGENT_12_Video_Producer.md`](system/agents/AGENT_12_Video_Producer.md).
