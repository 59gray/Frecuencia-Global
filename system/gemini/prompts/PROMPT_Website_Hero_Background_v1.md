# PROMPT — Website Hero Background v1

**Fecha:** 2026-04-02  
**Solicitante:** Codex  
**Sistema:** Frecuencia Global / Gemini Visual Creator  
**Destino:** Website hero background / editorial key visual

## Objective

Generar un visual panorámico para la portada del sitio de Frecuencia Global. Debe funcionar como ancla visual real para el primer viewport sin competir con el título principal ni ensuciar la legibilidad.

## Brand Constraints

- Fondo base `#0A0A0F`
- Acento dominante `#00E5FF`
- Acento secundario `#FF00E5` en proporción menor
- Look dark editorial, institucional, contemporáneo
- ADN electrónico sutil, no festival ni sci-fi genérico
- Sin texto, sin logos embebidos, sin banderas saturadas
- Componentes permitidos: grid sutil, líneas de señal, nodos, overlays de datos, mapas abstractos, infraestructura crítica
- Componentes prohibidos: globo genérico, audífonos, tornamesas, bocinas, equalizers, rave crowd, cyberpunk caricaturesco

## Composition Notes

- Imagen landscape ancha para uso web
- Reservar el tercio izquierdo con negativo visual y contraste controlado para overlay de titular
- Concentrar la acción visual en el tercio derecho y zona media
- La imagen debe sugerir red global, infraestructura, señal y tensión geopolítica
- Debe sentirse premium y reusable para otras piezas web/OG

## Primary Prompt

```text
Generate a wide cinematic editorial background image for the website hero of "Frecuencia Global", a media brand about geopolitics, culture, and technology.

COMPOSITION: very wide landscape image, calm dark negative space on the left 35% for headline overlay, visual action concentrated in the right 65%, layered depth, asymmetrical composition, no centered subject.
STYLE: Frecuencia Global dark editorial, institutional, contemporary, subtle electronic DNA, serious and premium, more world-affairs magazine than cyberpunk poster.
LIGHTING: low-key night lighting, cyan signal glow as dominant light source, restrained magenta secondary reflections, controlled contrast, soft atmospheric bloom.
COLOR: base #0A0A0F deep black, dominant accent #00E5FF electric cyan, restrained accent #FF00E5 neon magenta, optional tiny hints of #1A1A2E surface tones, no colors outside the Frecuencia Global palette.
TEXTURE: subtle monochrome grain, thin grid at very low opacity, smooth dark gradients, clean surface quality, no noisy clutter.
DEPTH: layered foreground and midground, atmospheric haze, soft focus background, feeling of global systems and hidden infrastructure.
ELEMENTS_ALLOWED: abstract world network routes, submarine cable paths, data overlays, signal nodes, thin waveform lines, map fragments, control-room ambience, satellite-style topographic patterns, subtle HUD accents.
ELEMENTS_PROHIBITED: text, logos, captions, flags, generic globe icon, headphones, turntables, speakers, equalizers, DJ booths, festival lights, anime sci-fi, gaming UI, exaggerated glitch.
ASPECT_RATIO: wide landscape for website hero, high resolution.
LEGIBILITY_PRIORITY: left side must remain darker and visually calmer for white text overlay, right side can carry more detail but must not become chaotic.
BRAND_ALIGNMENT: Frecuencia Global, a geopolitics platform with subtle club-energy aesthetics; the image must feel like global tension, data flow, and cultural signal without becoming generic tech wallpaper.

Create a polished, publication-ready hero image with no text embedded.
```

## Variation Prompt

```text
Same brief, but push the concept toward submarine cables, command-center interfaces, and hidden network chokepoints rather than a cityscape. Keep the left side quieter and the right side denser. No text.
```
