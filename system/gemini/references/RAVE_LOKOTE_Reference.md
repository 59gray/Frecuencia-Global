## ESTÉTICA GENERAL

RAVE LOKOTE representa la intersección entre:
- Energía de rave (intensidad, pulso, comunidad)
- Producción electrónica mexicana (identidad local + alcance global)
- Actitud contracultural (no mainstream, no complaciente)

### Cómo se traduce a Frecuencia Global:
- La **intensidad** se canaliza como **urgencia informativa** (no como fiesta)
- El **pulso electrónico** se convierte en **ritmo editorial** (datos que fluyen)
- La **actitud** se traduce en **autoridad no convencional** (diferente a medios tradicionales)

## PRINCIPIOS DE PRODUCCIÓN

### 1. Bass como fundamento
- El bajo no es decoración — es la base estructural
- Sub bass (30-60 Hz) presente pero controlado
- Mid bass (60-200 Hz) define el carácter de cada pieza
- **Aplicación FG:** el bajo transmite gravedad y presencia

### 2. Percusión precisa
- Kicks procesados, no genéricos
- Hi-hats con movimiento y variación sutil
- Snares/claps con espacio y reverb controlado
- **Aplicación FG:** percusión que marca tiempo editorial, como un reloj geopolítico

### 3. Síntesis con intención
- Cada sonido sintetizado tiene un propósito narrativo
- Pads para crear atmósfera (no para rellenar)
- Leads para momentos de énfasis (datos clave, "drops" informativos)
- **Aplicación FG:** los sintetizadores son la "voz" de los datos

### 4. Espacio y respiro
- No todo suena al mismo tiempo
- Arrangement que respira — tensión y liberación
- Silence is a tool
- **Aplicación FG:** dejar espacio para voiceover es prioridad absoluta

### 5. Texturas electrónicas
- Glitch, noise y artifacts como textura, no como error
- Risers y sweeps para transiciones
- Foley procesado electrónicamente
- **Aplicación FG:** las texturas crean la sensación de "interceptar una señal"

## PERFIL SONORO POR PILAR

> **Nota:** estos perfiles son adaptativos. Deben evolucionar con el contenido.

### Geopolitik Drop
- **Referencia RAVE LOKOTE:** máxima intensidad controlada
- **BPM:** 120-135
- **Carácter:** tenso, preciso, impactante
- **Elementos clave:** sub bass build → controlled drop, percussion militarista, leads metálicos

### Bass & Borders
- **Referencia RAVE LOKOTE:** fusión cultural, exploración
- **BPM:** 130-145
- **Carácter:** energético, multicultural, explorador
- **Elementos clave:** bass warped/wonky, percusión étnica procesada, texturas orgánicas + digitales

### Frecuencia Global
- **Referencia RAVE LOKOTE:** flow rítmico, pulso constante
- **BPM:** 125-140
- **Carácter:** ágil, informativo, rítmico
- **Elementos clave:** arpeggios digitales, hi-hats en movimiento, bass clean

### Behind the Policy
- **Referencia RAVE LOKOTE:** versión ambient/downtempo
- **BPM:** 90-110
- **Carácter:** reflexivo, analítico, profundo
- **Elementos clave:** pads profundos, percusión mínima, piano/keys procesados, sub drones

## SONIC BRANDING ELEMENTS

### Logo sound (sonic logo)
- Duración: 1-3 segundos
- Elementos: frequency sweep + sub impact + resonant tail
- Debe evocar: "sintonizando una frecuencia" → "señal capturada"
- Variantes: full (para intros), micro (para transiciones)

### Pillar signatures
Cada pilar tiene un sonido de 0.5-1s que lo identifica:
- Geopolitik Drop: radar ping + bass swell
- Bass & Borders: bass drop + percussion hint
- Frecuencia Global: data sweep + rhythm pulse
- Behind the Policy: low drone + clean key note

### Transition stings
- In/out de segmentos: frequency sweep + filter movement
- Cambio de tema: glitch + silence + new element entry
- "Drop" informativo: riser → impact + bass
