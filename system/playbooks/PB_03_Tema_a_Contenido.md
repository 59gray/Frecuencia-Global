# PLAYBOOK 03 — Convertir Tema en Reel / Carrusel / Post

**Sistema:** Frecuencia Global  
**Tipo:** Procedimiento operativo  
**Versión:** 1.0  
**Última actualización:** 2026-03-30

---

## CUÁNDO USAR ESTE PLAYBOOK

Cuando ya tienes un tema definido (con o sin investigación completa) y necesitas convertirlo en una pieza publicable de formato corto: reel/TikTok, carrusel de Instagram, o post estático.

---

## DECISIÓN DE FORMATO

Antes de producir, elige el formato según el tema y el objetivo:

| Si el tema es... | Formato recomendado | Por qué |
|------------------|---------------------|---------|
| Dato sorprendente o revelador | **Reel/TikTok** (60-90s) | Impacto visual + viralidad |
| Proceso con pasos o comparación | **Carrusel IG** (5-8 slides) | Estructura visual secuencial |
| Opinión / análisis corto | **Post estático** + caption largo | Profundidad sin video |
| Noticia rápida con contexto | **Reel** (30-60s) | Velocidad + engagement |
| Lista / ranking | **Carrusel** (1 ítem por slide) | Swipe engagement |
| Explicación de concepto | **Carrusel** o **Reel** | Depende de complejidad visual |

## RUTA A: TEMA → REEL / TIKTOK

### Paso 1: Estructura del guión (15 min)

```
[HOOK: 0-3 seg]
UNA frase que detenga el scroll.
Técnicas: pregunta provocadora, dato impactante, contradicción, cuenta regresiva.

[CONTEXTO: 3-15 seg]
¿Qué pasa? ¿Por qué debería importarme?
Máximo 2 frases.

[DESARROLLO: 15-50 seg]
2-3 datos clave con ritmo rápido.
1 dato por cada 10-15 segundos.
Incluir indicaciones visuales: [MAPA], [GRÁFICO], [B-ROLL].

[CIERRE: 50-60/90 seg]
Dato final de impacto o pregunta abierta.
CTA: "Sígueme para más" / "¿Qué opinas? Comenta."
```

### Paso 2: Producción visual (20-30 min)

0. Si la pieza es video, pasar primero por el Video Producer Agent para generar storyboard, shot list, editing brief y cover brief
1. Abrir template de Reel según pilar:
   - Geopolitik Drop → overlay cian
   - Bass & Borders → overlay magenta
   - Frecuencia Global → overlay verde ácido
   - Behind the Policy → overlay azul
2. Grabar o seleccionar footage
3. Insertar textos en pantalla (Space Grotesk Medium, fondo semitransparente)
4. Respetar safe areas (150px sup/inf, 60px laterales)
5. Añadir lower third con metadata (JetBrains Mono)
6. Pill de pilar en esquina superior
7. Exportar: 1080×1920 MP4, <50MB

### Paso 3: Caption y metadata (10 min)

```
Copy: [Frase hook o resumen del tema en 1 línea]
[Contexto expandido en 2-3 frases — valor adicional que no está en el video]
[CTA: pregunta que invite a comentar]

Hashtags (máx 5 relevantes):
#GeopolíticaExplicada #RelacionesInternacionales #FrecuenciaGlobal #[TemaEspecífico] #[PilarTag]
```

## RUTA B: TEMA → CARRUSEL DE INSTAGRAM

### Paso 1: Estructura del contenido (15 min)

```
SLIDE 1 — COVER
Título provocativo: máximo 8 palabras.
Subtítulo con contexto: máximo 12 palabras.
Pill de pilar. Frequency line decorativa.

SLIDE 2 — CONTEXTO
¿Por qué importa este tema? 1 párrafo corto (20-30 palabras).

SLIDES 3-6 — CONTENIDO CENTRAL
1 idea por slide. Máximo 40 palabras por slide.
Estructura por slide:
- Dato o afirmación clave (Bebas Neue, grande)
- Explicación o insight (Space Grotesk, más pequeño)
- Fuente o metadata si aplica (JetBrains Mono, micro)

SLIDE 7 — CIERRE / TAKEAWAY
"¿Qué significa esto?" — Resumen en 1-2 frases.
O una reflexión/pregunta que cierre.

SLIDE 8 — CTA
Logo FG + handles.
"Guarda este post" / "Comparte si te sorprendió" / "Comenta: ¿qué opinas?"
```

### Paso 2: Producción visual (30-45 min)

1. Abrir template de carrusel del pilar (v4)
2. Por cada slide:
   a. Insertar texto exacto del guión
   b. Aplicar jerarquía tipográfica
   c. Background: DarkSolid o DarkGrid
   d. Color de acento según pilar
   e. Elementos decorativos: signal nodes, frequency lines
3. Cover: más visual, con título grande
4. Cierre: logo y CTA prominentes
5. Verificar continuidad visual entre slides
6. Exportar: 1080×1350 PNG por slide

### Paso 3: Caption y metadata (10 min)

```
Copy: [Título del carrusel]
[2-3 frases expandiendo el valor del carrusel]
[CTA: "Desliza para ver los datos →" o "Guarda para leer después"]

Hashtags (máx 10 para IG):
#[Tema] #[Pilar] #FrecuenciaGlobal #Geopolítica #AnálisisInternacional
#[3-5 hashtags específicos del tema]
```

## RUTA C: TEMA → POST ESTÁTICO

### Paso 1: Estructura (10 min)

```
IMAGEN:
Diseño informativo simple:
- Título central (Bebas Neue)
- 1-3 datos o puntos clave (Space Grotesk)
- Pill de pilar
- Background del sistema
- Logo FG

CAPTION (250-500 palabras):
Párrafo 1: Hook — abrir con dato o pregunta
Párrafo 2-3: Desarrollo — contexto y análisis
Párrafo 4: Cierre — implicación o reflexión
CTA: Pregunta abierta a la audiencia
```

### Paso 2: Producción (20 min)

1. Template: post cuadrado 1080×1080 o portrait 1080×1350
2. Diseño limpio, centrado en texto
3. Exportar PNG
4. Escribir caption largo con valor editorial

## REPURPOSING — 1 TEMA, MÚLTIPLES FORMATOS

Si un tema tiene suficiente material, produce múltiples formatos:

```
TEMA FUENTE
    ├── Reel/TikTok (30-60s) — dato más impactante
    ├── Carrusel IG (5-8 slides) — análisis estructurado
    ├── Post estático IG — resumen visual
    ├── Thread X (8-12 tweets) — narrativa secuencial
    └── Story IG — encuesta o pregunta sobre el tema
```

**Regla de repurposing:** cada formato debe aportar valor independiente. No es copiar-pegar: es adaptar ángulo, tono y estructura al formato.

**Regla adicional para video:** todo clip derivado debe tener hook, CTA y editing brief propios; no basta con cortar el master y exportar.

---

## CHECKLIST ANTES DE ENVIAR A QA

```
□ Formato correcto para el tema
□ Pilar asignado y color consistente
□ Hook efectivo (primer slide / primeros 3 seg)
□ Datos verificados (provienen de ficha Research)
□ Tono alineado con pilar
□ CTA presente y específico
□ Resolución correcta (1080×1920 reel / 1080×1350 carrusel / 1080×1080 post)
□ Naming: FG_[PILAR]_[FORMATO]_[TEMA]_[VERSION]
□ Caption y hashtags escritos
□ Enviado con brief original y ficha de research
```
