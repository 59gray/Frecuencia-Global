Frecuencia Global — paquete v3 listo para Canva

Incluye:
- 6 portadas para highlights de Instagram
- 4 covers de series
- 1 reel cover master
- 1 guía visual del reel cover

Uso:
1. Highlights:
   - Sube cada PNG como portada del highlight correspondiente.
   - Si quieres simplificar, quédate con: News, Policy, Maps, About.

2. Covers de series:
   - Úsalos como portada base para carrusel, reel cover exportado o post de anuncio.
   - Sustituye solo el texto central; respeta márgenes y jerarquía.

3. Reel cover master:
   - Usa la versión normal para trabajar.
   - Usa la guía solo para verificar que el titular y tags queden dentro de la zona segura.
