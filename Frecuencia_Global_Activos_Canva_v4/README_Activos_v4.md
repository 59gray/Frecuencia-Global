Uso recomendado:
1. Sube cada PNG a Canva como base.
2. Duplica la pieza y reemplaza únicamente:
   - titular
   - subheadline
   - metadata
   - imagen, mapa o gráfico
   - texto modular interno
3. Mantén márgenes, jerarquías y pesos.
4. No cambies el color estructural por serie.
